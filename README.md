# PoolCRM
 
# API Эндпоинты Pool CRM

## 🔐 Аутентификация и главная страница

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Главная страница |
| GET | `/login` | Страница входа |
| POST | `/login` | Обработка входа (email/пароль) |
| GET | `/logout` | Выход из системы (удаление сессии) |
| GET | `/register` | Страница регистрации/подачи заявки |
| POST | `/register` | Обработка заявки от родителя |

## 📊 Дашборд

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/dashboard` | Панель управления (статистика, дети для родителя) |

## 👶 Дети

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/children` | Список детей (фильтр по роли) |
| GET | `/children/add` | Страница добавления ребёнка |
| POST | `/children/add` | Добавление ребёнка (только родитель) |
| GET | `/children/{child_id}/edit` | Страница редактирования ребёнка |
| POST | `/children/{child_id}/edit` | Редактирование ребёнка |
| GET | `/children/{child_id}/delete` | Удаление ребёнка (только админ) |

## 👥 Группы

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/groups` | Список групп (фильтр по роли) |
| GET | `/groups/add` | Страница создания группы (админ) |
| POST | `/groups/add` | Создание группы (админ) |
| GET | `/groups/{group_id}/edit` | Страница редактирования группы (админ) |
| POST | `/groups/{group_id}/edit` | Редактирование группы (админ) |
| GET | `/groups/{group_id}/delete` | Удаление группы (админ) |

## 🏊 Тренеры

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/trainers` | Список тренеров (админ) |
| GET | `/trainers/add` | Страница добавления тренера (админ) |
| POST | `/trainers/add` | Создание тренера (админ) |
| GET | `/trainers/{coach_id}/edit` | Страница редактирования тренера (админ) |
| POST | `/trainers/{coach_id}/edit` | Редактирование тренера (админ) |
| GET | `/trainers/{coach_id}/delete` | Удаление тренера (админ) |

## 📅 Тренировки

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/trainings` | Список тренировок (с фильтрацией по группе и статусу) |

## ✅ Посещаемость

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/attendance` | Страница посещаемости (выбор тренировки) |
| POST | `/attendance/save` | Сохранение отметок посещаемости (тренер/админ) |

## 📝 Заявки (админ)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/applications` | Список заявок (фильтр по статусу) |
| GET | `/applications/{app_id}` | Детали заявки |
| GET | `/applications/{app_id}/approve` | Одобрение заявки (создание родителя/ребёнка/зачисления) |
| GET | `/applications/{app_id}/reject` | Отклонение заявки |
| GET | `/applications/{app_id}/waiting` | Перевод в лист ожидания |
| POST | `/applications/{app_id}/comment` | Добавление комментария к заявке |

## 🔄 Переводы

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/transfers` | Список запросов на перевод |
| GET | `/transfers/add` | Страница создания запроса (тренер) |
| POST | `/transfers/add` | Создание запроса на перевод (тренер) |
| GET | `/transfers/{transfer_id}/approve` | Одобрение перевода (админ) |
| GET | `/transfers/{transfer_id}/reject` | Отклонение перевода (админ) |

## 👤 Профиль родителя

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/profile` | Профиль родителя (с генерацией VK кода) |
| POST | `/profile` | Обновление профиля родителя |
| GET | `/profile/unlink-vk` | Отвязка VK аккаунта |

## 🔌 API эндпоинты для VK бота

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/parents/{parent_id}/generate-vk-code` | Генерация 6-значного кода для привязки VK (действует 10 минут) |

## 👥 Роли и права доступа

| Роль | Доступ |
|------|--------|
| **admin** | Полный доступ ко всем административным функциям |
| **parent** | Свои дети, профиль, просмотр тренировок и посещаемости |
| **coach** | Свои группы, тренировки, отметка посещаемости, создание запросов на перевод |
